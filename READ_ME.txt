---------------------INSTRUCTIONS TO RUN THIS APPLICATION-----------------------------------
1. Run the following commands in R before running this application
	install.packages("shiny")
	install.packages("ggplot2")
	install.packages("arules")
	install.packages("gdata")
	install.packages("e1071")
	install.packages("Cairo")

2. Then copy the "ui.R" "server.R" "data.csv" in a seperate folder called "myapp" and load the files in R Studio to run the application
